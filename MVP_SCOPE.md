# MVP Scope - Shoraj Learning Platform

**Goal**: Launch a working learning platform that teaches fundamentals deeply. Quality and Trust over Scale.

## 1. Core Modules
### ✅ Module 1: Courses (FOUNDATION)
-   **Features**: Course list, Course detail, Module-based structure, Text + Video, Progress tracking.
-   **Out**: Certificates, Forums, Live classes.

### ✅ Module 2: Tests & Assessments (CRITICAL)
-   **Features**: Standalone & Course-linked tests, MCQs + Numerical, Time-bound/Untimed, Immediate Feedback (Reasoning).
-   **Philosophy**: Tests expose gaps.

### ✅ Module 3: Learning Paths (LIGHT)
-   **Features**: Curated sequences (e.g., "Statistics Foundations").

### ✅ Module 4: Teacher Presence
-   **Features**: Philosophy page, Instructor bio, Clear subject ownership.

## 2. User System
-   Sign up / Login.
-   Dashboard: Enrolled courses, Progress, Test attempts.

## 3. Payments
-   Platform ready for paid content (schema supported).
-   **MVP**: "Enroll" gating logic internal. No external payment gateway required for launch (optional).

## 4. Admin
-   Internal content management (Seed scripts or direct DB access/Admin API).

## 5. Content Scope (Launch)
-   1 Flagship Course (e.g., Statistics Foundations).
-   1-2 Diagnostic Tests.
-   1 Learning Path.

## 6. Technical Principles
-   Monorepo.
-   Simple Backend (Auth, Content, Tests).
-   Clear API boundaries.

## 7. Explicitly Out of Scope
-   Mobile App, AI Tutors, Forums, Gamification, Certificates.
